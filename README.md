# 📊 Customer Shopping Behavior Analysis

## 📌 Overview
This project focuses on analyzing customer shopping behavior to extract meaningful insights that help improve business decisions.

The goal is to understand **customer preferences, purchasing patterns, and key factors influencing buying decisions** such as discounts, ratings, and demographics.

---

## 📂 Dataset
- 📊 **Total Records:** 3,900 transactions  
- 📑 **Features:** 18 columns  

### 🔍 Key Data Includes:
- Customer demographics (Age, Gender, Location)
- Purchase details (Item, Category, Amount, Season)
- Behavioral data (Discount, Reviews, Frequency, Subscription)

### ⚠️ Data Issues:
- Missing values in **Review Rating** column handled during preprocessing  

---

## 🛠️ Tools & Technologies
- 🐍 Python (Pandas, NumPy, Matplotlib, Seaborn)
- 🛢️ SQL (PostgreSQL / MySQL / SQL Server)
- 📊 Power BI
- 📑 Gamma AI (Presentation)
- 🧾 Jupyter Notebook

---

## 🔄 Project Steps

### 1. Data Loading & Cleaning (Python)
- Imported dataset using Pandas  
- Handled missing values (median imputation)  
- Standardized column names  
- Performed feature engineering:
  - Age groups
  - Purchase frequency  

### 2. Exploratory Data Analysis (EDA)
- Analyzed data distributions  
- Identified trends in:
  - Customer segments  
  - Product categories  
  - Spending behavior  

### 3. Database Integration
- Loaded cleaned data into SQL database  
- Structured tables for efficient querying  

### 4. Data Analysis (SQL)
Performed business queries such as:
- Revenue by gender  
- High-spending discount users  
- Top-rated products  
- Customer segmentation  
- Revenue by age group
